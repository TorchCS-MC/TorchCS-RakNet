
# TorchCS RakNet

---

TorchCS RakNet is a network protocol for games, such as Minecraft Bedrock Dedicated.
It allows you to integrate your game project as a game protocol or use it for developing proxies or custom server software.
TorchCS RakNet was programmed in C to ensure compatibility with almost any programming language.
